# Portfolio v2 (Next.js)

The repository now contains only the v2 portfolio application built with Next.js. The entire app lives in `v2/`.

## Getting started
- `cd v2`
- `npm install`
- `npm run dev` to start the app, or `npm run build` / `npm start` for production
- `npm run lint` to run ESLint
